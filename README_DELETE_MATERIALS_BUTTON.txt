PB JOBHUB - DELETE MATERIALS BUTTON

Added to Material Costs page:
- Delete Material Cost Entries
- Delete Imported PDF Checklist Material Lines

Safety:
- Material cost delete requires typing DELETE MATERIALS.
- Imported material delete requires typing DELETE IMPORTED MATERIALS.
- Deleting a material cost entry does not delete the product from the product list.
- Deleting an imported material line does not delete the job.

How to use:
1. Open Material Costs.
2. Scroll to Delete Material Cost Entries.
3. Select the wrong/duplicate rows.
4. Type DELETE MATERIALS.
5. Click Delete Selected Material Cost Entries.

For imported PDF material lines:
1. Scroll to Imported PDF Checklist Material Lines.
2. Select the wrong imported rows.
3. Type DELETE IMPORTED MATERIALS.
4. Click Delete Selected Imported PDF Material Lines.
